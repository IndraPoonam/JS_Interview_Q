React.js Interview Questions for Freshers Answer: React.js ek JavaScript library hai jo user interfaces banane ke liye use hoti hai. Isko Facebook ne develop kiya tha. React components ke concept par based hai jo UI ke reusable pieces hote hain. Answer: React component ek JavaScript function ya class hota hai jo HTML ko render karta hai. Components ko reuse kiya ja sakta hai aur yeh apni internal state manage kar sakte hain. Components do types ke hote hain: functional aur class components. Answer:Functional Components: Yeh simple JavaScript functions hote hain jo JSX return karte hain. Hooks ke aane ke baad, yeh bhi state aur lifecycle methods handle kar sakte hain. Class Components: Yeh JavaScript classes hoti hain jo React.Component se extend karti hain aur render method implement karti hain jo JSX return karti hain. Answer: JSX (JavaScript XML) ek syntax extension hai jo React ke sath use hoti hai. Yeh JavaScript ko HTML jaise syntax mein likhne deti hai, jo components ko define karte waqt easy aur readable hota hai. JSX ko compile karke JavaScript mein convert kiya jata hai. Answer: State: State ek component ka local data hota hai jo component ke andar manage hota hai aur change ho sakta hai. Props: Props (short for properties) ek component ko pass kiye gaye data hote hain jo parent component se aate hain. Props read-only hote hain aur child component unko change nahi kar sakta.
